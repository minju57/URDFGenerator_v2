# URDF Generator 

이 프로젝트는 Gradio를 기반으로 제작된 URDF(Unified Robot Description Format) 생성 및 시각화 도구입니다. 

## 주요 기능
* 로봇 모델 URDF 파일 생성
* 직관적인 웹 UI (Gradio) 제공

## 로컬 실행 방법
\`\`\`bash
pip install -r requirements.txt
python app.py
\`\`\`
